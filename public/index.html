            Why HttpOnly + Secure? HttpOnly blocks JavaScript
            access to sensitive cookies, and Secure sends cookies only over
            HTTPS.
          
    
          
    
            Session vs persistent cookies? Session cookies
            expire when the browser closes; persistent cookies remain until
            their explicit expiry date.
          
    
          
    
            If localStorage is full? A QuotaExceededError is
            thrown. Handle with try/catch, keep payloads small, and clean old
            entries.
          
    
          
    
            When use cookies over localStorage? When the server
            must read auth/session data on each request.
          
    
          
    
            Risk of passwords in sessionStorage? XSS can still
            steal them, so passwords should never be stored in browser storage.
          
    
          
    
            Incognito behavior Storage usually exists only
            during the private window and is wiped when it closes.
